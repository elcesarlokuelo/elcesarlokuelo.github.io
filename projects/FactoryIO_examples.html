Factory IO Examples

Factory IO · PLC · Ladder · Codesys · TIA Portal

This page contains a collection of solved examples for the Factory IO software, which is a 3D simulation platform for industrial automation. The examples cover various scenarios and challenges, and provides solutions using different PLC programming environments such as Codesys and TIA Portal and different communication protocols like Modbus, Profinet or OPC-UA.

All project content is available in the repository: https://github.com/elcesarlokuelo/Gaita_electronica_MIDI

Each example will include a description of the problem, the implemented solution in the chosen PLC code language, and the corresponding software and communication protocol used. For all the examples there will be a video demonstration of the solution in action within the Factory IO simulation environment.

Software

The microcontroller was programmed in Arduino IDE using C++. MIDI communication was handled via USB using Hairless MIDI Serial and LoopMIDI. Between those two softwares, I can translate the USB-serial messages emmited by the Arduino into the baud rate and format expected by the MIDI protocol, and route them to the synthesizer.

The synthesizer used for sound generation was Universal Piper, a standalone bagpipe synthesizer with a library of 16 different bagpipe types including the Galician bagpipe besides others like the Scottish or Irish, configurable via MIDI Control Change messages.

Mechanical Design

Parts were modelled in FreeCAD and printed in PLA using an Anycubic Kobra 2 printer. The 3D-printed enclosure houses all the electronics and keeps the controls accessible to the player, though the ergonomics leave room for improvement in future iterations.
